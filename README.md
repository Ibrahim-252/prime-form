# Prime Form — Executive Health Coaching

Prime Form is a premium, high-end health coaching platform designed specifically for busy professionals and corporate executives. It combines science-backed protocols with a luxury digital experience to deliver measurable results in energy, longevity, and physical performance.

## 🚀 Tech Stack

- **React + Vite**: High-performance frontend framework and build tool.
- **Tailwind CSS**: Utility-first styling for a responsive, modern layout.
- **Framer Motion**: Advanced animations for a cinematic, premium user experience.
- **Lucide React**: Clean and professional iconography.
- **Custom Design System**: "Deep Obsidian" aesthetic featuring glassmorphism, bento grids, and high-end dark mode tokens.

## ✨ Key Features

- **Cinematic Hero**: Local video background with "Deep Obsidian" glassmorphism overlays.
- **Bento Grid Benefits**: Modern, non-linear layout showcasing the core value propositions.
- **Executive Portal**: (In development) A dedicated area for clients to track their progress and blueprints.
- **Mobile Optimized**: Fully responsive design that looks stunning on any device.

## 🛠️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## 📽️ Asset Requirements

- The hero background video should be placed in `video/hero-bg.mp4`.
- Brand logos are located in the `/logo` directory.
